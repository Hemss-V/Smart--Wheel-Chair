The Smart Wheelchair is an IoT-based assistive mobility system designed to enhance safety, accessibility, and independence for physically challenged individuals.
The system integrates bluetooth based voice control, manual control, and real-time obstacle detection to ensure safe and intelligent navigation.

Features : 
- Voice-Based Control – Users can control movement using voice commands.
- Manual Control – Alternative physical control mode for flexibility.
- Automatic Obstacle Avoidance – Detects obstacles in real-time and prevents collisions by stopping
